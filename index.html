Insider Web SDK 測試站
首頁 商品頁 購物車 完成頁

首頁

此站為靜態頁面骨架，用於與 Insider Integration Wizard 對齊：頁面事件、使用者屬性、商品／購物車／完成頁等後續可逐步接上。

頁面地圖

  • 商品頁 — Product page events
  • 購物車 — Cart page events
  • 訂單完成頁 — Confirmation page events
提醒：在後台「測試 Insider Tag 整合」前，請先在此專案各頁 <head> 內貼上 Insider 提供的標籤程式碼。

本機預覽：在專案根目錄執行靜態伺服器後開啟首頁即可。
